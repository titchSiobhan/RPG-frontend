# Questage 

A lightweight roguelike RPG text‑based sim where you create a character, explore the realm, complete quests, fight monsters, and survive a world driven by luck, events, and dynamic encounters.

## Features 

- React and TypeScript UI

- Shop and inventory

- Name characters

- Player dashboard

- Quest log

- Mini pop up messages

- Exploration events

- API integration with backend

- Responsive layout

## Stack

- React
- TypeScript
- CSS 
- Fetch API for backend communication

## Setup 

**Prerequisites**

- Node.js
- npm
- Backend server running (Barely Social backend)

## Install dependencies

`npm install`


## Start Server

`npm run dev`

**Server**

Runs on `http://localhost:5173`
